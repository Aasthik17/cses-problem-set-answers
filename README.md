# CSES Problem Set – Solutions

This repository contains my **solutions to the CSES Problem Set**, a well-known collection of competitive programming problems used for **DSA practice**, **interview preparation**, and **algorithmic thinking**.

All solutions are written with a focus on:
- Correctness  
- Optimal time & space complexity  
- Clear logic and readability  

---

## About CSES

The **CSES Problem Set** is a curated collection of algorithmic problems covering a wide range of topics such as:
- Sorting & Searching  
- Dynamic Programming  
- Graph Algorithms  
- Trees  
- Mathematics  
- String Algorithms  

Official site: https://cses.fi/problemset/
